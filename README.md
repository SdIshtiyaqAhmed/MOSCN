## Getting Started

To explore or run the code:

1. Clone the repository:
- git clone https://github.com/SdIshtiyaqAhmed/MOSCN.git
- cd MOSCN
2. Open the assignment documents to understand the problem statements and requirements.  
3. Compile and run the corresponding C++ source files using a C++17-compatible compiler, adjusting file paths as needed:
- g++ -std=c++17 <path-to-source-file>.cpp -o program
- ./program
